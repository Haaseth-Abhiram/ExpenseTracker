# Expense Tracker (Java)

## Overview
This project implements a console-based Expense Tracker system using Java.
It helps users manage their daily expenses by allowing them to add, view,
search, calculate, and remove expenses efficiently.

The project is designed as a beginner-friendly Java application
to practice Object-Oriented Programming concepts and data handling using ArrayList.

---

## Problem Statement
Managing daily expenses manually can be difficult and unorganized.

This project provides a simple solution to:
- Store expense details
- Categorize expenses
- Calculate total spending
- Search expenses easily
- Remove unwanted expense records

---

## Features
- Add new expenses
- View all expenses
- Calculate total expenses
- Search expenses by category
- Remove expenses using ID
- Menu-driven console interface
- Automatic expense ID generation

## How the Application Works
1. User selects an option from the menu
2. Application performs the requested operation
3. Data is stored temporarily using ArrayList
4. User can continue operations until exit

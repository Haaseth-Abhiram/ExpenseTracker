# Expense Tracker (Java)

A simple console-based Expense Tracker application developed using Java.

------------------------------------------------------------
FEATURES
------------------------------------------------------------

- Add new expenses
- View all expenses
- Calculate total expenses
- Search expenses by category
- Remove expenses by ID
- Menu-driven console application
- Uses ArrayList and OOP concepts

------------------------------------------------------------
TECHNOLOGIES USED
------------------------------------------------------------

- Java
- ArrayList
- Scanner Class
- Object-Oriented Programming (OOP)

------------------------------------------------------------
PROJECT STRUCTURE
------------------------------------------------------------

com.version
│
├── Expense.java
└── ExpenseTracker.java

------------------------------------------------------------
HOW TO RUN
------------------------------------------------------------

1. Clone the repository

git clone https://github.com/your-username/Expense-Tracker-Java.git

2. Open the project in any Java IDE

- Eclipse
- IntelliJ IDEA
- VS Code

3. Compile and run ExpenseTracker.java
